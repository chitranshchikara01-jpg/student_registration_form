# student_registration_form

A desktop-based Student Registration Form application developed using Python and Tkinter. This application allows users to enter student information through a graphical user interface (GUI) and securely store the data in a database for future reference and management.

## Features

- User-friendly GUI built with Tkinter
- Student registration and data entry
- Input validation for accurate data collection
- Store student records in a database
- View and manage registered student information
- Simple and efficient database integration

- ## Student Information Collected

The application can store details such as:

- ID 
- Name
- Last Name
- DOB
- Gender
- Course
- Email
- Password
